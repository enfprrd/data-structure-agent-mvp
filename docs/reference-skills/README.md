# Reference Skills

This folder stores external agent skills copied into the project as local reference material.

## style-extractor

Source: https://github.com/Lucent-Snow/style-extractor

Use this reference when extracting reusable web visual language from a site or web app, including:

- colors, typography, spacing, and layout patterns
- component states and interaction rules
- motion timing, keyframes, and runtime animation evidence
- evidence-backed style guide outputs

Entry point: `style-extractor/SKILL.md`
